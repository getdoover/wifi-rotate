## Overview

Cycle through multiple WiFi networks with configurable hold times for each network.

<br/>

## Configuration

| Setting | Description | Default |
|---------|-------------|---------|
| **Profiles** | List of WiFi profiles with SSID and password | `Required` |

<br/>
## Integrations

This is a standalone app with no dependencies on other Doover apps.
